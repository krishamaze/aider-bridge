🎯 QUICK START (3 STEPS):

1️⃣  START API SERVER
    Windows: scripts\start-api.bat
    PowerShell: .\scripts\start-api.ps1
    
    Should see: Running on http://127.0.0.1:5000

2️⃣  INSTALL EXTENSION
    • Go to chrome://extensions/
    • Enable "Developer mode"
    • Click "Load unpacked"
    • Select the "extension" folder

3️⃣  USE IT!
    • Open claude.ai or perplexity.ai
    • Ask Claude to format response as YAML
    • Click Aider Bridge extension
    • Click "Run Aider Bridge"
    • Watch it auto-fill! ✨
